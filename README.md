# Click.ru Tools

Инструменты для работы с [Click.ru API](https://help.click.ru/6634) — платформы для управления рекламными кампаниями. Репозиторий содержит скрипты для автоматизации рутинных задач маркетингового агентства.

## 📂 Состав

| Скрипт | Назначение |
|--------|-----------|
| `daily_budget_report.py` | Мониторинг бюджетов Яндекс.Директ: прогноз остатка и алерты по кабинетам |

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/antonbru/clickru-tools.git
cd clickru-tools
```

### 2. Настройте конфигурацию

```bash
cp config.example.json config.json
```

Отредактируйте `config.json` — укажите ваш **мастер-токен** Click.ru API:

```json
{
    "clickru": {
        "token": "ваш_токен_clickru_api",
        "base_url": "https://api.click.ru/V0"
    }
}
```

> **Как получить токен:** зайдите в [Click.ru](https://click.ru) → Профиль → API → сгенерируйте мастер-токен.

Или задайте токен через переменную окружения:

```bash
export CLICKRU_TOKEN="ваш_токен"
```

### 3. Запустите отчёт

```bash
# Только проблемные кабинеты (запас < 3 дней)
python daily_budget_report.py

# Все кабинеты, включая с нормальным бюджетом
python daily_budget_report.py --all

# С явным указанием пути к конфигу
python daily_budget_report.py -c /путь/к/config.json
```

## 🔧 Как работает

1. **Авторизация:** скрипт получает ID мастер-пользователя через `GET /user`
2. **Обход суб-пользователей:** получает список всех USER-аккаунтов через `GET /users`
3. **Сбор балансов:** для каждого пользователя:
   - `GET /accounts` → фильтрует активные кабинеты Яндекс.Директ
   - `GET /users/payers` → суммирует балансы плательщиков
4. **Статистика расходов:** через `GET /stat/v2`:
   - Основной расчёт — по вчерашнему дню
   - Если вчера расхода не было (например, выходной) — среднее за 7 дней
5. **Прогноз:** `(баланс_кабинета + баланс_плательщиков) / дневной_расход`
6. **Алерты:**
   - 🔴 ≤ 0 дней — деньги закончились
   - 🟠 < 1 дня — критически мало
   - 🟡 < 3 дней — требуется пополнение
   - 🟢 ≥ 3 дней — норма (скрыто по умолчанию)

## ⚙️ Автоматизация (cron)

Для ежедневного запуска добавьте в crontab:

```bash
# Каждый день в 9:00 по Москве
0 6 * * * cd /путь/к/clickru-tools && python daily_budget_report.py >> /var/log/clickru-report.log 2>&1
```
